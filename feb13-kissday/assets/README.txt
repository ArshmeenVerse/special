# Audio File Instructions for Kiss Day

To make the sound effects work, please download and place the following audio files into the `assets` folder:

1. **Background Music** -> `bg-music.mp3` (Soft romantic instrumental)
2. **Kiss Sound** -> `kiss.mp3` (Subtle kiss sound)
3. **Heartbeat** -> `heartbeat.mp3` (Slow heartbeat sound)
4. **Sparkle/Magic** -> `sparkle.mp3` (Optional: Magical chime sound)

You can find free sound effects on sites like Pixabay or Freesound.
